# Welcome to CampusConnect - Your Campus Social Network! Welcome to the Campus Social Network project! This platform is designed to connect students on campus, allowing them to interact, share information, and build a vibrant campus community. ## Getting Started These instructions will help you set up and run the project on your local machine for development and testing purposes. You can also deploy it to a live system. ### Prerequisites Before you begin, ensure you have met the following requirements: 1. **Python**: Ensure that Python is installed on your computer, and it should be version 3.7 or higher. If you don't have Python installed, you can download it from the official website: [Python Downloads](https://www.python.org/downloads/). 2. **Django**: Django is the web framework used for this project. You can install Django using pip, which typically comes with your Python installation. Open your terminal or command prompt and run: ```shell pip install django ``` 3. **Git**: Git is a version control system. If you don't have Git installed, you can download it from the official website: [Git Downloads](https://git-scm.com/downloads). ### Cloning the Repository To get started with the project, follow these steps: 1. Open your terminal or command prompt. 2. Navigate to the directory where you want to store the project. 3. Use the following command to clone the project's repository: ```shell git clone https://github.com/ninijoe/campusconnect ``` 4. After cloning the repository, create a new branch to work on your changes. It's important not to work on the main branch to avoid disrupting the workflow. You can create a new branch with a name of your choice using the following command: ```shell git checkout -b your-branch-name ``` Make sure to replace `your-branch-name` with a descriptive name for your branch. 5. To confirm that you are on your newly created branch, you can run: ```shell git branch ``` Now, you're ready to start developing and contributing to the project. If you have any questions or need guidance on development, feel free to reach out to the project maintainers. ### Running the Project Once you have successfully cloned the repository and created your branch, follow these steps to run the project on your local machine: 1. Navigate to the project directory: ```shell cd campusconnect ``` 2. Run the following command to start the development server: ```shell python manage.py runserver ``` Replace `python` with your specific Python version if necessary. 3. After running the command, you should see output indicating that the development server is running. By default, the project will be accessible at `http://127.0.0.1:8000/` in your web browser. You're now ready to explore, develop, and test the CampusConnect project locally. If you encounter any issues or have questions, don't hesitate to ask for assistance. Happy coding!
